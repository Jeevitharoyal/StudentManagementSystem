# StudentManagementSystem
## Description
This is a console-based Student Management System developed in Java. It allows users to manage student records using CRUD operations and stores data using file handling and serialization.

## Features
- Add Student
- Display All Students
- Search Student
- Update Student
- Delete Student
- Save Students
- Load Students

  ## Technologies Used
  - Java
  - Eclipse IDE
  - OOP Concepts
  - ArrayList
  - File Handling
  - Serialization

    ## Project Structure
    - Main.java
    - Student.java
    - StudentManager.java
    - FileManager.java
   
      ## How to Run
      1.Clone this repository.
      2.Open the project in Eclipse IDE.
      3.Run 'Main.java'.
      4.Use the menu to manage student records.
